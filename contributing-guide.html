This wiki is based on Wiki.js and its user guide can be found here. When creating new pages, if in doubt, go for the "Visual Editor" option as it's the easiest and more natural way to edit content 

Everyone with an account can currently:

  • Create new pages
  • Edit existing pages
  • Comment and respond to comments

Currently, content moderators can also:

  • Move pages
  • Delete pages
  • Edit the navigation on the left

If you are interested in becoming a content moderator or have feedback on the above permissions - get in touch.
